# Local Agent Platform

A portable, Docker-based platform for running local AI agents with Ollama.

## Goals

- Run on Ubuntu 24.04, Debian 13, and macOS
- Add agents through modular configuration
- Restrict agent access to the host operating system
- Monitor CPU, memory, containers, and agent activity
- Evaluate agent inputs, outputs, and task performance
- Keep configuration portable across machines

## Planned Components

- Ollama
- Agent API and orchestrator
- Agent dashboard
- PostgreSQL
- Prometheus
- Grafana
- Container monitoring
- Agent evaluation and tracing
